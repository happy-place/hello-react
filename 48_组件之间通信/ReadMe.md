## 48_组件之间通信
```aidl
组件之间关系 
    父子组件
    兄弟组件
    祖孙后代组件（跨级）

组件之间通信手段：
    直接传递属性和回调函数：props。父->子（属性）,子->父(回调函数)
    消息订阅发布：pub-sub、event。谁接收谁订阅，谁传递谁发布。
    集中式状态管理：redux、dva。存在公共位置，谁用谁取。
    context: 声明在公共区域、借助 Provider 传递信息，谁使用谁声明绑定 contextType，然后从 this.context 中获取信息。

常用解决方案：
    父子组件：props
    兄弟组件：消息订阅发布、集中式状态管理
    跨级组件：消息订阅发布、集中式状态管理、context(使用较少，常用与封装插件)

```
